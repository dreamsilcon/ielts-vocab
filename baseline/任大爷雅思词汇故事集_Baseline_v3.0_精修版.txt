《任大爷雅思词汇故事集》Baseline v3.0（精修版）

目标：
词汇表 → 故事 → 人物 → 世界观 → 长期记忆

核心理念：
高密度 + 强语境 + 可复读 + 可记忆

------------------------------------
一、核心目标
------------------------------------
词汇 + 阅读 + 音标 + 词性 + 中文释义 + 语境记忆

------------------------------------
二、写作原则
------------------------------------

原则1：100%覆盖
- 写前必须词表审计
- 写后必须覆盖审计
- Missing ≠ 0 不允许结束

原则2：禁止硬塞词
- 先设计故事结构
- 再自然嵌入词汇
- 不允许“词汇拼接感”

原则3：自然语境优先
- 所有词必须出现在合理语境
- 不写例句式文本

原则4：高密度表达（新增）
- 减少废话
- 保持信息量
- 每句话必须有意义

------------------------------------
三、段落结构（核心升级）
------------------------------------

强制规则：

一个 Part =

【一整段英文】
+
【一整段中文】

禁止：
- 一句话一行
- 碎片化段落

目标：
- 像雅思阅读文章
- 可连续阅读
- 可整段背诵

------------------------------------
四、词汇出现规则
------------------------------------

- 每个核心词：至少出现 1 次
- 尽量自然出现（不重复堆砌）

------------------------------------
五、跨章节复现机制（记忆增强）
------------------------------------

- 每章复现 3–5 个旧词
- 不标粗
- 作为隐性复习

------------------------------------
六、英文格式
------------------------------------

核心词必须：

**word** (/IPA/)

示例：
**ecosystem** (/ˈiːkəʊˌsɪstəm/)

------------------------------------
七、中文格式
------------------------------------

规则：

- 不写“中文翻译”
- 紧跟英文段落

核心词标注：

中文（英文，词性，音标）

示例：
生态系统（ecosystem，n. 生态系统，/ˈiːkəʊˌsɪstəm/）

优化规则（新增）：
- 第一次出现：完整标注
- 后续出现：可弱化或不标

------------------------------------
八、音标规则
------------------------------------

- 统一 IPA
- 禁止图片音标
- 禁止非标准拼音

------------------------------------
九、故事结构
------------------------------------

词量决定结构：

30词 → 3幕
40词 → 4幕
50词+ → 主线 + Epilogue

------------------------------------
十、世界观规则
------------------------------------

Level 2（标准）：

- 同一世界观（Academy Universe）
- 可出现旧角色
- 不强制连续剧情

------------------------------------
十一、角色系统（新增）
------------------------------------

固定角色：

- Leo（学习者）
- Sophia（执行者/领导者）
- Elias（科学家）

每章：
- 可新增1个角色

目的：
- 建立记忆锚点

------------------------------------
十二、写作强化（IELTS）
------------------------------------

每章可附：

Key Expressions（简短）

例如：
- play a crucial role in
- pose a significant threat to

------------------------------------
十三、漏词处理
------------------------------------

发现漏词：

必须增加：

Epilogue 或 Appendix

自然补齐

------------------------------------
十四、最终交付标准
------------------------------------

内容：
✓ 故事完整
✓ 可阅读
✓ 无拼接感

语言：
✓ 加粗词汇
✓ IPA完整
✓ 中文完整

结构：
✓ 一段英文 + 一段中文
✓ 无碎片化

质量：
✓ 无漏词
✓ 无降质

审计：
Coverage: 100%
Missing: 0

------------------------------------

中文（整段）
