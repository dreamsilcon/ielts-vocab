《任大爷雅思词汇故事集》Baseline v2.0

目标：
把《宋鹏昊雅思核心词汇精讲精练》的每一章：
词汇表 → 故事 → 人物 → 世界观 → 长期记忆

一、核心目标
词汇 + 阅读 + 音标 + 词性 + 中文释义 + 语境记忆

二、写作原则

原则1：100%覆盖
- 开始写作前必须进行词表审计
- 写作完成后必须进行覆盖率审计
- Missing ≠ 0 不得宣布完结

原则2：禁止硬塞单词
- 先有故事，再放词
- 根据词汇结构设计故事框架
- 保持自然语境

原则3：优先自然语境
- 单词必须出现在合理情节中
- 不写孤立例句式故事

三、英文格式
所有核心词必须：
- 加粗
- 附带 IPA 音标

格式：
**entrepreneur** (/ˌɒntrəprəˈnɜːr/)

四、中文格式
规则：
- 英文段落下面直接跟中文
- 不写“中文翻译：”
- 核心词必须标注：
  中文释义 + 英文 + 词性 + 音标

格式：
企业家（entrepreneur，n. 企业家，/ˌɒntrəprəˈnɜːr/）

五、音标规则
统一采用 IPA 国际音标

六、故事长度
- 30词左右：3幕结构
- 40词左右：4幕结构
- 50词以上：主线 + Epilogue

七、世界观原则
优先形成连续宇宙

Chapter 02
The Green Valley Academy

Chapter 03
The Academy Through the Ages

如条件允许，持续沿用 Academy Universe 世界观。

八、覆盖审计
章节结束必须提供：

Chapter XX Audit

Coverage: 100%
Missing: 0

九、漏词处理规则
发现漏词时：
- 增加 Epilogue
或
- 增加 Appendix

自然补齐全部词汇。

十、最终交付标准

内容：
✓ 故事完整
✓ 可独立阅读
✓ 不像单词表

语言：
✓ 所有核心词加粗
✓ IPA完整
✓ 中文完整
✓ 词性完整

质量：
✓ 无硬塞词
✓ 无漏词
✓ 无降质段落

审计：
✓ Coverage = 100%
✓ Missing = 0
